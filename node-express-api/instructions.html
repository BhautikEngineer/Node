Create a Basic REST API using Node.js and Express

This example will demonstrate how to set up routes, handle HTTP requests, and send JSON responses.

  1. Create a new directory for your project and navigate into it:
    2. mkdir node-express-api
cd node-express-api
  2. Initialize a Node.js project and install the required dependencies:
    3. npm init -y
npm install express
  3. Create an app.js file with the following content:
    4. <script type="text/javascript">
const express = require('express');
const app = express();
const port = 3000;

// Sample data
const products = [
  { id: 1, name: 'Product 1' },
  { id: 2, name: 'Product 2' },
];

// Middleware to parse JSON requests
app.use(express.json());

// Route to get all products
app.get('/products', (req, res) => {
  res.json(products);
});

// Route to get a specific product by ID
app.get('/products/:id', (req, res) => {
  const id = parseInt(req.params.id);
  const product = products.find(p => p.id === id);
  
  if (product) {
    res.json(product);
  } else {
    res.status(404).json({ error: 'Product not found' });
  }
});

// Start the server
app.listen(port, () => {
  console.log(`Server is listening on port ${port}`);
});
</script>
  4. Run the Express app:
    5. node app.js
  5. Open your web browser or use a tool like cURL to test the API:
    • Get all products: 
      http://localhost:3000/products
    • Get a specific product by ID (replace :id with an actual product ID): 
      http://localhost:3000/products/:id

In this example, we created an Express app with two routes: one to get all products and another to get a specific product by its ID. The app uses middleware to parse JSON requests, and it sends JSON responses containing the product data.

Remember that this is a basic example to introduce you to building APIs with Express. In real-world applications, you would likely connect to a database, add authentication, and handle more complex scenarios.
